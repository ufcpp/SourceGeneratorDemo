# DependencyProperty and INotifyPropertyChanged Generators

This project provides two Source Generators for WPF and MVVM development:

1. **DependencyPropertyGenerator** - Generates WPF DependencyProperty boilerplate
2. **NotifyPropertyChangedGenerator** - Generates INotifyPropertyChanged implementation

Both generators use partial properties to provide a clean, declarative syntax.

These generators address common pain points in WPF development. Many developers have requested such tools, and numerous implementations exist across the community. This project demonstrates how straightforward it is to create these generators using modern C# Source Generator capabilities.

## DependencyPropertyGenerator

### Usage

Add the `[DependencyProperty]` attribute to a partial property in a class that inherits from `DependencyObject`:

```csharp
using System.Windows;
using DependencyPropertyGenerator;

public partial class MyControl : DependencyObject
{
    [DependencyProperty]
    public partial int Count { get; set; }

    [DependencyProperty]
    public partial string? Title { get; set; }
}
```

### Generated Code

The generator creates the DependencyProperty field and implements the property accessors:

```csharp
// <auto-generated/>
#nullable enable
#pragma warning disable

using System.Windows;

partial class MyControl
{
    public static readonly DependencyProperty CountProperty =
        DependencyProperty.Register(
            nameof(Count),
            typeof(int),
            typeof(MyControl),
            new PropertyMetadata(default(int)));

    public partial int Count
    {
        get => (int)GetValue(CountProperty);
        set => SetValue(CountProperty, value);
    }

    public static readonly DependencyProperty TitleProperty =
        DependencyProperty.Register(
            nameof(Title),
            typeof(string),
            typeof(MyControl),
            new PropertyMetadata(default(string)));

    public partial string Title
    {
        get => (string)GetValue(TitleProperty);
        set => SetValue(TitleProperty, value);
    }
}
```

## NotifyPropertyChangedGenerator

### Usage

Add the `[NotifyPropertyChanged]` attribute to a partial property:

```csharp
using NotifyPropertyChangedGenerator;

public partial class ViewModel
{
    [NotifyPropertyChanged]
    public partial int Count { get; set; }

    [NotifyPropertyChanged]
    public partial string? Name { get; set; }
}
```

### Generated Code

The generator creates backing fields, implements `INotifyPropertyChanged`, and adds change notification logic:

```csharp
// <auto-generated/>
#nullable enable
#pragma warning disable

using System.ComponentModel;

partial class ViewModel : INotifyPropertyChanged
{
    private static readonly PropertyChangedEventArgs CountPropertyChangedEventArgs = new PropertyChangedEventArgs(nameof(Count));

    private int _count;

    public partial int Count
    {
        get => _count;
        set => SetProperty(ref _count, value, CountPropertyChangedEventArgs);
    }

    private static readonly PropertyChangedEventArgs NamePropertyChangedEventArgs = new PropertyChangedEventArgs(nameof(Name));

    private string _name;

    public partial string Name
    {
        get => _name;
        set => SetProperty(ref _name, value, NamePropertyChangedEventArgs);
    }

    public event PropertyChangedEventHandler? PropertyChanged;

    protected virtual void OnPropertyChanged(PropertyChangedEventArgs e)
    {
        PropertyChanged?.Invoke(this, e);
    }

    protected bool SetProperty<T>(ref T storage, T value, PropertyChangedEventArgs e)
    {
        if (global::System.Collections.Generic.EqualityComparer<T>.Default.Equals(storage, value))
        {
            return false;
        }
        storage = value;
        OnPropertyChanged(e);
        return true;
    }
}
```
